# Data_Mining
# Thuật toán DBSCAN 
- Là thuật toán dùng để gom cụm dữ liệu, được đưa ra vào năm 1996 bởi nhóm tác giả. 
- Thuật toán phân cụm không gian dựa trên mật độ với nhiễu.
- Nó có thể xử lý mạnh mẽ dữ liệu có nhiễu.
- Thuật toán không cần chọn trước số cụm cần chia như K-Means, nhạy cảm với khoảng cách, dùng khoảng cách Euclid để tính khoảng cách giữa các điểm với nhau.
